การแจ้งเตือน

💡 ในแอปจะเด้ง toast อัตโนมัติ · ถ้าอนุญาต browser notification จะได้รับเมื่อ tab อยู่ background ด้วย
💾
สำรองข้อมูล (Backup)
ดาวน์โหลดข้อมูลทั้งหมดเป็นไฟล์ — กู้คืนได้เมื่อต้องการ
💡 แนะนำ: สำรองข้อมูลทุกสัปดาห์ หรือก่อนทำการแก้ไขสำคัญ
📁 ไฟล์ที่ได้เป็น JSON ขนาดเล็ก — เก็บใน Google Drive/OneDrive/โฟลเดอร์ในเครื่อง
🔐
Sync รายชื่อสำหรับหน้าสมัคร
อัพเดท public/staff_list เพื่อให้สมาชิกใหม่สมัครได้ (จำเป็นหลังเปลี่ยน security rules)
💡 ทำครั้งเดียวพอ: กดปุ่มนี้หลังอัพเกรดระบบใหม่ เพื่อ copy รายชื่อทีมงานไป public path
🔒 ปลอดภัย: copy แค่ ชื่อ + ชื่อเล่น + ประเภทงาน (ไม่ copy ธนาคาร/เมล/ข้อมูลส่วนตัว)
⚙️ หลังกดครั้งแรก: ระบบจะอัพเดท public list อัตโนมัติทุกครั้งที่เพิ่ม/แก้/ลบทีมงาน
